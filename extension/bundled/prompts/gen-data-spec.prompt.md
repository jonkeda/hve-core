# Generate Data Spec

Invoke the gen-data-spec agent to generate comprehensive data dictionaries, machine-readable data profiles, and objective summaries for downstream analysis.

## Inputs

* ${input:dataSource}: (Required) Path to data file or directory to profile.
* ${input:outputDir}: (Optional) Output directory for generated specs. Defaults to a location adjacent to the data source.
